# React Hola Mundo ⚛️

## Installation.

```bash
git clone https://github.com/EduardYan/react-hola-mundo.git
cd react-hola-mundo
```

### Dependecies

```bash
corepack enable
corepack prepare pnpm@latest --activate
pnpm i
```

**or using npm:**

```bash
npm install -g npm@latest
npm i
```

## Run

```bash
pnpm run dev
```

**or using npm:**

```bash
npm run dev
```
